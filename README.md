Hotel Nebula - Praticando MongoDB
Criei uma modelagem do banco de dados do Hotel Nebula com alguns dados fictícios para simular o funcionamento do hotel e adicionei consultas básicas para praticar meus novos conhecimentos em MongoDB.

🛠️ O que foi feito:
1. Modelagem de Dados
Organizei as informações em documentos JSON, utilizando os conceitos de objetos embutidos e arrays para facilitar as buscas. As principais coleções criadas foram:

Hóspedes e Funcionários: Dados de contato, cargos e preferências.

Quartos: Tipos de quartos e suas comodidades (banheira, vista, etc).

Reservas e Hospedagens: Controle de datas, valores e consumo extra.

2. Implementação
Os dados foram inseridos no MongoDB Compass utilizando o formato Extended JSON, garantindo que datas e valores decimais fossem registrados corretamente.

3. Consultas Práticas (Queries)
Para testar o banco, realizei consultas que respondem perguntas do dia a dia do hotel, como:

Quais quartos estão disponíveis com "vista para o mar"?
Quais hóspedes são frequentes no hotel?

Objetivo: Este projeto serviu para consolidar meu aprendizado inicial em bancos NoSQL, focando em flexibilidade de esquemas e performance de consultas.
